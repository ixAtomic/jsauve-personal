Phase 2

I then began determining which tools I would like to use for the job. I decided on the following:

  1. VueJs
  2. Vuetify
  3. Directus*
  4. Docker*
  5. Postgresql*
  6. Raspberry Pi*

the tools with the asterisk have been deprecated because I found a quicker way to retrieve content. I am now just pulling content directly from a json object.


The reason for the change was that the maintenance of running my own backend required more time than I had to spend and using static content will be easier to maintain for the small scale of this website.
